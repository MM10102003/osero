## 概要
C++で作成したオセロゲームです。  
プレイヤーとコンピュータが対戦できる簡易的な対戦型プログラムです。

## 機能
- オセロの基本ルールに基づいた石の配置判定
- 置ける場所の判定処理
- 石のひっくり返し処理
- ランダムによるコンピュータの手生成
- 盤面表示

## 使用技術
- C++
- 標準ライブラリ（iostream, cstdlib, ctime）

## 工夫した点
- 8方向すべてに対してひっくり返し判定を行う処理を実装
- 置ける手の有無を事前に判定し、パス処理を実装
- コンピュータとの対戦機能を実装

## 実行方法
Visual Studio等でビルドし実行してください。
